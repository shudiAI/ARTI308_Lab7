# ARTI308_Lab7
In this project, we worked on a fake advertising dataset to predict whether a user will click on an advertisement.

We performed the following steps:

Imported and explored the dataset using pandas (info() and describe()).
Conducted exploratory data analysis using seaborn (histograms, jointplots, and pairplots) to understand relationships between features.
Selected key numerical features such as:
Daily Time Spent on Site
Age
Area Income
Daily Internet Usage
Male
Split the data into training and testing sets using train_test_split.
Built a Logistic Regression model using scikit-learn.
Trained the model and made predictions on the test data.
Evaluated the model using a classification report, achieving around 90% accuracy.
Improved convergence by increasing max_iter when needed.

Overall, the model performed well in predicting whether a user would click on an ad based on their behavior and demographic features.
